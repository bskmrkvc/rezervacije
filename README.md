# rezervacije
Jednostavan sistem rezervacija karata u C++ jeziku kao mini-projekat iz predmeta OBJ1

Ispunjava sledece uslove:
• bar 3 klase  
• nasleđivanje  
• statičke članove  
• inspektore  
• mutatore  
• preklapanje operatora  
• string  
• ili aray ili list ili vector  
• ili pair ili map ili multimap  
• stack  
• ili queue ili deque  
• ili sort ili qsort  
• datoteku  
• šablon (template)
